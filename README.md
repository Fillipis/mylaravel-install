# mylaravelapp
Projeto público de estudos com o framework [Laravel].

# minhas configurações de máquina
Editor de texto VS Code, Xampp (3.3.0), MariaDB (15.1), Gitbash e Composer instalados.

# política de versionamento (github)
Passo 1: entrar no seu diretório local, exemplo: <pre>C:\xampp\htdocs\</pre>
Passo 2: clonar o projeto [MYLARAVELAPP]: <pre>git clone git@github.com:Fillipis/mylaravelapp.git</pre>

# metodologia de versionamento (gitflow)
Passo 1: entrar no diretório do projeto clonado: <pre>C:\xampp\htdocs\mylaravelapp\</pre>
Passo 2: iniciar o gitflow: <pre>git flow init</pre>
Passo 3: deixar as branch's por padrão/opcional do gitflow: <pre>[master] e [develop]</pre>
Passo 4: deixar os prefixos por padrão/opcional do gitflow: <pre>[feature], [hotfix] e [...]</pre>
